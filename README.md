Community Health Awareness Platform

Overview

The Community Health Awareness Platform is a web-based system designed to improve public access to health information and promote health awareness within communities.

Features

- User Registration and Login
- Disease Awareness Information
- Health Tips
- Emergency Healthcare Contacts
- Interactive Health Quiz
- Quiz Result Tracking
- Admin Dashboard
- Disease Management
- Health Tips Management
- Emergency Contact Management

Technologies Used

- HTML
- CSS
- PHP
- MySQL
- XAMPP
- Visual Studio Code

Installation

1. Download or clone the repository.
2. Copy the project folder into the XAMPP htdocs directory.
3. Import the health_awareness.sql database into phpMyAdmin.
4. Start Apache and MySQL in XAMPP.
5. Open the project in a browser.

SDG Alignment

This project supports Sustainable Development Goal 3 (Good Health and Well-Being) by promoting health awareness and public access to health information.
